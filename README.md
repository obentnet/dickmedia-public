# DickMedia
一个简易的视频拼接处理软件。

# 如何使用?
你需要自行下载ffmpeg到程序目录下。  
然后运行launcher.py

Content Auth Server -> Auth Server Back Auth Infomation -> 

# 功能列表
* √ 拼接 模式1
* √ 拼接 模式2
* √ 5秒视频切割 清水素材
* √ 标题随机乱序
* √ 音频分贝提升
* √ 开发者模式
